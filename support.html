FAQ

How to get started?

Download and install the CoTerm app, then follow the in-app instructions to complete the setup. You'll need an AI service API Key (such as Anthropic, OpenAI).

Which platforms are supported?

CoTerm supports iOS, macOS, Windows, and Linux platforms. Mobile and desktop can be used synchronously.

Do I need an API key?

Yes, you need to configure your own AI service API Key to use AI features. We support multiple providers including Anthropic (Claude), OpenAI (GPT-4), etc.

What does Pro subscription include?

Pro subscription provides unlimited AI sessions, unlimited Agent execution time, multi-session workspace management and other advanced features. Subscriptions are available through the App Store with monthly or annual billing.
